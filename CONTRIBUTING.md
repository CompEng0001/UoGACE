# Contribution Guidlines
## How to contribute

Follow the steps to merge you branch with the main branch. This needs to be done every Friday.

First you want to 
``` bash
git checkout -YOURBRANCH-

git add
git commit -m " YOURE MESSAGE "
git push

git checkout main
git pull
git merge -YOURBRANCH-

```
